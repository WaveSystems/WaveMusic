= WaveMusic


Overall status: <b>30%</b>

== About the project

<i>WaveMusic</i> is a school project for cloud music storage and streaming API, this application will allow users to sign up, store their music and listen it everywhere once the mobile clients are released. This API will be private but free to sign up and consume it as a developer. Until the project is under 70% of development you will not be able to use it, but after that feel free to use it to your own purposes or consume it after the first release.

== Installation

  git clone https://github.com/WaveSystems/WaveMusic.git
  cp config/database.yml.sample config/database.yml # Add your postgres configuration
  bundle install
  rake db:migrate

And enjoy!
